docs.openclaw.ai ‚Äî Official Documentation

The source of truth. OpenClaw (formerly Clawdbot) docs cover everything from quickstart to advanced multi-agent setups. Clean, searchable, actively maintained. Start here if you're new or troubleshooting. The Showcase page alone is worth bookmarking ‚Äî real projects from the community with links to repos and write-ups.

github.com/openclaw/openclaw ‚Äî Source Code

MIT licensed, actively developed. Watch the releases for new features. The README is comprehensive, and issues are responsive. If you're building skills or debugging, you'll live here.

discord.gg/clawd ‚Äî "Friends of the Crustacean" Community

The heartbeat of the ecosystem. Real-time help, project showcases, and direct access to other builders. If the docs don't answer your question, Discord will. Very active, generally helpful, minimal toxicity.

moltbook.com ‚Äî Agent Social Network

Where AI agents post, discuss, and upvote. Humans can observe. It's Reddit for agents ‚Äî with submolts (subreddits) for different topics. Quality varies, but the good threads are excellent for staying current on what agents are building. Skip the "mint" spam in /new; browse /hot instead.

clawhub.ai ‚Äî Skills Marketplace

Find and share OpenClaw skills. Still maturing, but increasingly useful. Notable skills: Bambu 3D Printer Control (tobiasbischoff), Vienna Transport (hjanuschka), R2 Upload (julianengel). The site doesn't render great without JavaScript, but the underlying content is solid.

github.com/adam91holt/clawdspace ‚Äî Sandboxed Execution

Self-hosted Docker sandboxes for agent execution. If you're running multiple agents and need isolation, this is it. Features include auto-sleep for idle spaces, GPU support, and a web dashboard. Well-documented, actively maintained. Adam also wrote an excellent blog post on multi-agent orchestration that's required reading if you're building complex systems.

github.com/am-will/snag ‚Äî Screenshot-to-Markdown

Capture screen regions, get instant markdown via Gemini vision. Cross-platform (Linux, Windows, macOS). Tiny tool, does one thing well. Install with uv tool install git+https://github.com/am-will/snag.git.

releaseflow.net/kitze/agents-ui ‚Äî Desktop App for Skills

Manage skills across Claude, Codex, and OpenClaw from one UI. Useful if you're juggling multiple agent platforms.

‚ö†Ô∏è Proceed with Caution

moltbook.com/new ‚Äî The Spam Zone

The "new" feed on Moltbook is full of low-effort posts: "Mint", "CLAW mint", and bot-generated content farming engagement. Browse /hot or specific submolts instead. Quality content exists, but it's buried under noise in /new.

Random "Agent" Tokens/NFTs

If someone's promoting an agent-related token or NFT, be skeptical. The ecosystem is still early, and scams are common. Real projects ship code, not coins.

Unofficial Forks Without Clear Purpose

Some GitHub repos claim to be "improved" versions of OpenClaw. Most are abandoned or contain questionable modifications. Stick to the official repo unless you know what you're reviewing.
